Коронавирусная инфекция в Беларуси

COVID-19 Монитор

Данные быстро меняются. В статистике может не быть информации о недавно зарегистрированных случаях.

Учтены только те, кто сдал тест. При подсчете случаев заболевания учитываются только пациенты, прошедшие диагностику с положительным результатом. Правила диагностики и возможность ее проведения зависят от страны. Информация по некоторым регионам может отсутствовать, поскольку данные ещё не были опубликованы или давно не обновлялись.

Почему данные из разных источников отличаются? Есть множество интернет-ресурсов, которые отслеживают и обрабатывают статистику распространения коронавирусной инфекции. При этом алгоритм сбора данных и интервал их обновления у каждого сайта разный.

    API для получения информации о коронавирусе

  • API root: /api/v1/
  • Projects list: /api/v1/projects/
  • Posts list: /api/v1/posts/
  • Locations list: /api/v1/locations/
  • Facts list: /api/v1/facts/
  • Photos list: /api/v1/photos/
